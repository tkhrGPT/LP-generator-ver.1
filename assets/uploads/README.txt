================================================================
  LP ジェネレーター — 画像アップロードフォルダ
================================================================

ここに LP 用の画像ファイルを入れてください。
Claude Code が画像の役割を自動判定し、LP の適切な位置に配置します。

------------------------------------------------------------
【ファイルの命名（任意）】

ファイル名の先頭にヒントを付けると、より正確に判定されます。

  hero-○○.jpg      → ヒーロー背景（横長・雰囲気重視）
  product-○○.jpg   → 商品メインビジュアル / コーチ写真
  lifestyle-○○.jpg → 使用シーン・生活感のある写真
  sns-○○.jpg       → SNS グリッド画像（正方形推奨）
  logo-○○.png      → ロゴ・バッジ・認定マーク
  bg-○○.jpg        → セクション背景テクスチャ

ヒントなしのファイル名でも Claude が視覚的に判定します。

------------------------------------------------------------
【対応フォーマット】

  JPEG (.jpg / .jpeg)  PNG (.png)  WebP (.webp)

------------------------------------------------------------
【配置のルール（参考）】

  hero     : 1枚だけ使用（複数あれば最初の1枚）
  product  : 1枚だけ使用
  lifestyle: 最大3枚。フィーチャー/ベネフィット差し込みに使用
  sns      : 最大6枚。SNS グリッドタイルに使用
  logo     : 複数可。trust 要素行に並べる
  background: 複数可。セクション背景 CSS に使用

------------------------------------------------------------
【画像がない場合】

  各ロールの画像がなければ CSS のグラデーション/カラーブロックで
  自動的に代替表示されます（プレースホルダーフォールバック）。

------------------------------------------------------------
このフォルダ内の画像は output/index.html からの相対パスで参照されます。
（../assets/uploads/ファイル名）

生成後、どの画像がどこに使われたかは
assets/image-manifest.md で確認できます。
